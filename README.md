# Nuxt project shop

example nuxt project shop

## Stack

- Nuxt 3
- Vue
- Typescript
- Tailwindcss
- lint
- prettier
- bun
- flowbite
- i18n
- Pinia

## Run

- dev $bun dev
- lint fix $bun lint:fix
- format $bun format

## Deploy

- build static .output/public $bun generate

- build server-side .output $bun build
- build preview prod $bun preview
